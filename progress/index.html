Progress Bars







	
<div class="progressbg">
	<div class="progress" style="width: 30%"></div>
</div>
<br>
<div class="progressbg">
	<div class="progress-alert" style="width: 40%"></div>
</div>
<br>
<div class="progressbg">
	<div class="progress-success" style="width: 50%"></div>
</div>
<br>
<div class="progressbg">
	<div class="progress-warning" style="width: 60%"></div>
</div>
<br>
<div class="progressbg">
	<div class="progress-info" style="width: 70%"></div>
</div>

There are 5 classes-

progress - A normal progress bar. (First one shown)

progress-alert - A red progress bar. (Second one shown)

progress-success - A green progress bar. (Third one shown)

progress-warning - A yellow progress bar. (Fourth one shown)

progress-info - A lighter shade of blue progress bar. (Last one shown)

You can customize how much of the bar is filled by changing the width css var for the inner div.



Animated Bars



	
<button onclick="animate_progress(document.getElementById('animated_bar'), 10, 30, 100)">Start Animation</button><br><br>
<div class="progressbg">
	<div id="animated_bar" class="progress" style="width: 30%"></div>
</div>


To use an animated progress bar, you need to import the MinimalistJS file as well. (See Getting Started). To animate the bar, call the following function with the following parameters in the following order-

animate_progress(elem, speed, start, end)

elem is the progress bar element.

speed is the how fast the progress bar should fill. The number is in terms of milliseconds. The speed is the wait time between each percent of the bar filled.

start is the percent at which to start.

end is the percent number at which to stop.



Custom Height



	
<div class="progressbg" style="height: 1px;">
	<div class="progress" style="width: 30%; height: 1px;"></div>
</div>
<br>
<div class="progressbg" style="height: 20px;">
	<div class="progress" style="width: 30%; height: 20px;"></div>
</div>

You can change the height of the progress bar by changing the CSS height variable. Be sure to change the height of both divs to make sure it renders properly.
