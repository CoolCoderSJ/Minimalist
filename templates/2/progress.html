There are 5 classes-

progress - A normal progress bar. (First one shown)

progress-alert - A red progress bar. (Second one shown)

progress-success - A green progress bar. (Third one shown)

progress-warning - A yellow progress bar. (Fourth one shown)

progress-info - A lighter shade of blue progress bar. (Last one shown)

You can customize how much of the bar is filled by changing the width css var for the inner div.



Animated Bars



	
<button onclick="document.getElementById('animated_bar').setAttribute('style', 'width: 30%; transition: 0.5s; transform: scaleX(3);')">Start Animation</button><br><br>
<div class="progressbg">
	<div id="animated_bar" class="progress" style="width: 30%"></div>
</div>


To use an animated progress bar, you need to use the javascript setAttribute function. To animate the bar, set the attribute 'style' with the following css-

transition: 0.5s - How fast you want it to move (Change 0.5s to any number)

transform: scaleX(3); is the CSS animation. (Change 3 to 100 divided by the decimal form of the percentage.)



Custom Height



	
<div class="progressbg" style="height: 1px;">
	<div class="progress" style="width: 30%; height: 1px;"></div>
</div>
<br>
<div class="progressbg" style="height: 20px;">
	<div class="progress" style="width: 30%; height: 20px;"></div>
</div>

You can change the height of the progress bar by changing the CSS height variable. Be sure to change the height of both divs to make sure it renders properly.
