Code



Did you know that you can copy text with CTRL + C or CMD + C?

		
/*

To use short code, such as the CTRL + C mentioned above, add a code class to an element.

	
		<code class="code"> CTRL + C</code>


Minimalist does not have direct support for syntax highlighting for codeblocks. To use syntax highlighting with codeblocks, visit Syntax Highlighter
